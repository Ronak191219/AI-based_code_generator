# 🤖 AI Code Generator

An AI-powered web application that generates Python code based on user input using Generative AI.

---

## 📌 Features

- Generate Python code from natural language prompts
- Simple and clean user interface
- Secure API key handling using environment variables
- Error handling for API limits and failures

---

## 🏗️ Architecture

User Input → Flask Backend → OpenAI API → Generated Code → UI Display  

---

## 🛠️ Tech Stack

| Layer      | Technology        |
|-----------|-----------------|
| Frontend  | HTML, CSS        |
| Backend   | Flask (Python)   |
| AI Model  | OpenAI API       |
| Env Mgmt  | python-dotenv    |
 

## ▶️ How to Run the Project

pip install -r requirements.txt
python app.py
